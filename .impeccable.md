## Design Context

### Users
Power users, developers, and customization enthusiasts running Windows with GlazeWM and Zebar. They use this in a desktop environment to monitor system stats, control media, and manage workspaces efficiently.

### Brand Personality
Technical, refined, customizable, unobtrusive.

### Aesthetic Direction
Modern desktop UI, supporting robust dark/light theming (like Catppuccin) and dynamic color generation. Needs to feel native yet distinctly polished and premium compared to standard Windows UI.

### Design Principles
1. **Density over whitespace**: This is a widget interface; information density matters more than generous padding.
2. **Contextual theming**: Colors and styles must adapt flawlessly to user-selected palettes and system themes.
3. **High contrast for glanceability**: System stats and critical indicators must be legible at a quick glance.
4. **Snappy interactions**: Because it's a system utility, UI must feel instant and lightweight, avoiding long animations.
